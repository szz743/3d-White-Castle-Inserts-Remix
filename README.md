# 3d-White-Castle-Inserts-Remix

This is a remix of excellent "White Castle with Matcha Insert / Organizer - Also Fits Promos" by js500, https://www.printables.com/model/1192543-white-castle-with-matcha-insert-organizer-also-fit

Original cards box has text on both sides, did not really need it, so I've made a version without that text. Text removal from SCAD file is easy, so I've just removed the text commands and made STL files, no other changes.

Printed using PLA by a printing service.
